# Drafters AI QA Documentation

## Overview

This repository contains the documentation required for AI-assisted QA and automation testing of the Drafters mobile application.

The purpose of this repository is to provide project knowledge, business rules, feature documentation, and testing guidelines that can be consumed by AI agents to generate, review, and maintain automation test cases.

## Repository Structure

```
.
├── AGENTS.md                # Main instructions for AI agents
├── flows/                   # Module-wise documentation
├── automation/              # Automation standards and guidelines
├── api/                     # API documentation
├── test-data/               # Test data and test accounts
└── reports/                 # QA report templates
```

## Documentation Status

- Initial repository structure created.
- Module documentation is under preparation.
- AI integration will be added in the next phase.

## Purpose

- Maintain project-specific QA documentation.
- Support AI-assisted automation testing.
- Standardize business rules and testing guidelines.
- Improve automation quality and consistency.
