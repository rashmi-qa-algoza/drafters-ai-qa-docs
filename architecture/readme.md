# Architecture Documentation

## Overview

This folder contains high-level architecture documentation for the Drafters platform.

Its purpose is to help QA Engineers, Developers, Business Analysts, Product Managers, Automation Engineers, and AI agents understand how the platform is structured before reviewing feature-level documentation.
